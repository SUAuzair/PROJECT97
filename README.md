
PROJECT97 is a native Android application developed using Kotlin. 
 Features

* 📲 User-friendly Android interface
* ⚙️ Built with modern Android development tools
* 🧩 Modular project structure
* 🔄 Gradle-based dependency management
* 📦 Scalable and maintainable codebase



 🛠️ Technologies Used

* **Kotlin** – Primary programming language
* **Android SDK** – Core development framework
* **Gradle (Kotlin DSL)** – Build system
* **XML** – UI design layouts
* **Android Studio** – Development environment

 ⚙️ Setup & Installation

 1. Clone the Repository

```bash
git clone https://github.com/SUAuzair/PROJECT97.git
```

### 2. Open in Android Studio

* Open **Android Studio**
* Click **Open**
* Select the folder:

```
PROJECT97-main/SYED
```

### 3. Sync Gradle

* Click **“Sync Project with Gradle Files”**

### 4. Run the App

* Connect an Android device OR start an emulator
* Click **Run ▶️**



 Requirements

* Android Studio (latest version recommended)
* Internet connection (for dependencies)
* Android SDK installed



Common Issues & Fixes

| Issue              | Solution                             |
| ------------------ | ------------------------------------ |
| Gradle Sync Failed | Check internet & retry               |
| SDK Missing        | Install required SDK via SDK Manager |
| App Not Running    | Start emulator or connect device     |
| Build Errors       | Clean & Rebuild project              |
